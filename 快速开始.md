## 🎯 三种使用方式

### 方式 1: 立即测试（推荐新手）

```powershell
# Windows PowerShell
$env:PYTHONUTF8=1
cd C:\Users\77012\Desktop\fishing-mcp-server
python test_local.py
```

看到 ✅ 测试完成就说明一切正常！

---

### 方式 2: 本地连接 Claude Code

1. **编辑 Claude Code 配置**
   
   打开 `~/.claude/settings.json`，添加：
   
   ```json
   {
     "mcpServers": {
       "fishing-game": {
         "command": "python",
         "args": ["C:/Users/77012/Desktop/fishing-mcp-server/server.py"],
         "env": {
           "PYTHONUTF8": "1"
         }
       }
     }
   }
   ```

2. **重启 Claude Code**

3. **测试**
   
   在 Claude Code 中说：
   ```
   帮我玩钓鱼游戏，先查看状态
   ```

## 🎮 游戏指令速查

| 指令 | 说明 | 示例 |
|------|------|------|
| `help` | 查看帮助 | `help` |
| `status` | 查看状态 | `status` |
| `cast [次数]` | 抛竿钓鱼 | `cast 10` |
| `buy <饵> <数量>` | 购买鱼饵 | `buy basic_worm 5` |
| `sell all` | 卖掉所有鱼 | `sell all` |
| `goto` | 查看地点列表 | `goto` |
| `goto <地点>` | 前往地点 | `goto misty_lake` |
| `shop` | 查看商店 | `shop` |
| `inventory` | 查看背包 | `inventory` |
| `encyclopedia` | 查看图鉴 | `encyclopedia` |

**高级技巧**：用 `;` 连接多条指令
```
buy basic_worm 10; cast 10; sell all
```

## 💡 游戏策略

1. **初期**：多抛竿，收集图鉴，首次发现奖励翻倍
2. **中期**：解锁新地点，探索稀有鱼
3. **高级**：用 `cast 10` 连钓省时间
4. **经济**：稀有鱼卖掉换点数，解锁新区域

## ❓ 常见问题

**Q: 中文显示乱码？**
A: Windows 用户运行前设置：`$env:PYTHONUTF8=1`

**Q: 存档在哪？**
A: 项目目录下的 `fishing_save.json`，删除它就重开

**Q: 怎么重开游戏？**
A: 删除存档文件，或让 Claude 调用 `fishing_new_game` 工具

**Q: 能多人一起玩吗？**
A: 目前是单人游戏，多人需要各自部署
