# 🚀 部署到 Render 完整教程（小白版）

## 前置准备

1. **注册 GitHub 账号**（如果没有）
   - 访问 https://github.com
   - 点击 Sign Up 注册

2. **注册 Render 账号**
   - 访问 https://render.com
   - 可以直接用 GitHub 账号登录

3. **安装 Git**（你电脑上已经有了）

---

## 步骤 1: 上传代码到 GitHub

### 1.1 在 GitHub 创建新仓库

1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 填写信息：
   - **Repository name**: `fishing-mcp-server`
   - **Description**: `钓鱼游戏 MCP 服务器`
   - **Public** 或 **Private** 都可以
   - ⚠️ **不要**勾选 "Add a README file"（我们已经有了）
4. 点击 "Create repository"

### 1.2 推送代码

打开 Windows Terminal 或 PowerShell，运行：

```powershell
# 进入项目目录
cd C:\Users\77012\Desktop\fishing-mcp-server

# 初始化 Git 仓库
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: 钓鱼游戏 MCP 服务器"

# 关联远程仓库（替换成你的 GitHub 用户名）
git remote add origin https://github.com/你的用户名/fishing-mcp-server.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

如果提示输入用户名和密码：
- 用户名：你的 GitHub 用户名
- 密码：需要用 **Personal Access Token**（不是登录密码）
  - 获取方式：GitHub → Settings → Developer settings → Personal access tokens → Generate new token

---

## 步骤 2: 在 Render 上部署

### 2.1 连接 GitHub 仓库

1. 登录 https://render.com
2. 点击右上角 **"New +"** 按钮
3. 选择 **"Web Service"**
4. 如果第一次使用，点击 **"Connect GitHub"** 授权
5. 选择你刚才创建的 `fishing-mcp-server` 仓库

### 2.2 配置服务

填写以下配置：

| 字段 | 值 |
|------|-----|
| **Name** | `fishing-mcp-server`（或你喜欢的名字） |
| **Region** | 选择 **Singapore** 或 **Oregon**（离中国近） |
| **Branch** | `main` |
| **Root Directory** | 留空 |
| **Runtime** | `Python 3` |
| **Build Command** | `pip install -r requirements.txt` |
| **Start Command** | `python app.py` |

### 2.3 选择套餐

- 选择 **Free**（免费套餐）
- 免费套餐限制：
  - 15 分钟无活动会自动休眠
  - 每月 750 小时运行时间（够用）
  - 512MB 内存

### 2.4 部署

1. 点击底部 **"Create Web Service"** 按钮
2. 等待部署（大约 2-5 分钟）
3. 看到绿色 **"Live"** 标志就表示部署成功了！

---

## 步骤 3: 获取服务器地址

部署成功后，你会看到一个地址，例如：

```
https://fishing-mcp-server-xxxx.onrender.com
```

**复制这个地址**，后面要用！

测试一下是否正常：
```
https://你的地址.onrender.com/health
```

应该返回：`{"status":"ok","service":"fishing-mcp-server"}`

---

## 步骤 4: 连接到 Claude Code

### 方法 1: 使用 /config 命令（推荐）

1. 在 Claude Code 中输入：`/config`
2. 找到 **MCP Servers** 部分
3. 添加新服务器

### 方法 2: 手动编辑配置文件

1. 打开配置文件：
   - Windows: `C:\Users\77012\.claude\settings.json`
   - 或者在 Claude Code 中输入：`! code ~/.claude/settings.json`

2. 添加以下配置：

```json
{
  "mcpServers": {
    "fishing-game": {
      "command": "python",
      "args": ["C:/Users/77012/Desktop/fishing-mcp-server/server.py"]
    }
  }
}
```

**注意**：这是本地连接方式。如果要远程连接 Render，等 MCP 支持 HTTP transport 后再更新配置。

3. 保存文件

4. 重启 Claude Code

## 常见问题

### Q: 部署后显示 "Build failed"
A: 检查 `requirements.txt` 和 `app.py` 是否正确上传到 GitHub

### Q: 服务一直显示 "Deploying"
A: 耐心等待 3-5 分钟，第一次部署会比较慢

### Q: 免费套餐够用吗？
A: 够用！免费套餐每月 750 小时，相当于 31 天 24 小时运行

### Q: 如何更新代码？
A: 修改本地代码后，运行：
```bash
git add .
git commit -m "更新说明"
git push
```
Render 会自动重新部署！

### Q: 如何查看日志？
A: Render 控制台 → 你的服务 → Logs 标签页
